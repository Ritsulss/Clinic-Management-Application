# Clinic Management Application

A simple CRUD (Create, Read, Update, Delete) web application designed by four students to manage **patients**, **doctors**, and **appointments** for a medical clinic. This project demonstrates basic full-stack development and database interaction using Python and HTML/CSS.

---

## Overview

This Clinic Management Application allows users to:

- Register and manage patient records  
- Add, view, and update doctor information  
- Schedule, edit, and delete appointments  

Itâ€™s a lightweight project ideal for learning how backend applications interact with a database and serve dynamic content to users.

---

## Features

**Patient Management:** Create, read, update, and delete patient information  
-**Doctor Management:** Handle doctor details  
- **Appointment Scheduling:** Schedule and manage patient appointments  
- Simple database storage using SQLite for persistence  
- Basic HTML and CSS for the front-end interface

---

## Tech Stack

This project uses:

- Python (Flask web framework)
- HTML / CSS for front-end pages
- Javascript for dynamic interactions
- SQLite database for data persistence

## Getting Started

Follow the steps below to set up this project locally:

### 1. Clone the repository  
```bash
git clone https://github.com/Ritsulss/Clinic-Management-Application.git
cd Clinic-Management-Application
```

### 2. Create a virtual environment (optional but recommended)
```bash
python -m venv venv
source venv/bin/activate  # macOS / Linux
venv\Scripts\activate     # Windows
```

### 3. Install Dependencies
```bash
pip install flask
```
### 4. Run the Application
```bash
python app.py
```
Open your browser and go to http://127.0.0.1:5000 to view the app.
